## THE NIRMAL FRAMEWORK - COMPLETE SPECIFICATION

### 1. THE TWO-ENTRY SYSTEM (Core Mechanic #1)

Every single trade has exactly two entry points:

ENTRY 1: Immediate entry at market price
  - Formula: Entry_1 = Current_Price
  - Timing: When technical setup confirms
  - Execution: Buy immediately

ENTRY 2: Averaging entry on dip
  - Formula: Entry_2 = Entry_1 × 0.92 (8% lower)
  - Timing: If price pulls back 8% or more
  - Execution: Buy at predetermined level

**Example:**
Stock: AMD
Current Price: $161
Entry 1: $161.00
Entry 2: $148.12 (8% lower)

### 2. THE TWO-TARGET SYSTEM (Core Mechanic #2)

Every trade has exactly two profit targets:

TARGET 1: Conservative exit
  - Formula: Target_1 = Entry_1 × 1.12 (12% gain)
  - Action: Sell 50% of total position
  
TARGET 2: Conviction hold  
  - Formula: Target_2 = Entry_1 × 1.25 (25% gain)
  - Action: Sell remaining 50% of position

**Real Example:**
AMD on June 21, 2024
Entry 1: $161
Entry 2: $152
Target 1: $175 (hit day 5) → Sell 50%
Target 2: $190 (hit day 12) → Sell remaining 50%
Total Return: +13.4% average
Status: ✅ WINNER

### 3. THE STOP-LOSS SYSTEM (Core Mechanic #3)

Stop losses are MANDATORY, NON-NEGOTIABLE.

STOP LOSS CALCULATION:
  Formula: SL = Entry_2 × 0.92 (8% below Entry 2)
  OR: SL = Support_Level × 0.98 (whichever is lower)

EXECUTION:
  - Place SELL order at SL level BEFORE entering trade
  - Automatic, no emotional decision
  - Hit SL = Execute exit immediately

**Risk/Reward Validation:**
  Minimum Required: Ratio ≥ 2.0 (at least 2:1)
  
  Example:
    Entry: $100
    Target 2: $128

## POSITION SIZING - THE 2% RULE

**RULE:** Never risk more than 2% of total portfolio on any single trade.

**Formula:**
Max_Risk_Amount = Portfolio_Value × 0.02
Position_Shares = Max_Risk_Amount / (Entry_1 - Stop_Loss)
Position_Value = Position_Shares × Entry_1

Cap at 5% max position value (never exceed)

**Example:**
Portfolio: $100,000
Entry 1: $150
Stop Loss: $135
Risk per share: $15

Max risk: $100,000 × 0.02 = $2,000
Shares: $2,000 / $15 = 133 shares
Position value: $19,950

Check 5% cap: $19,950 / $100,000 = 19.95% ❌ TOO BIG
Reduce to: 33 shares = $4,950 ✅ ACCEPTABLE

## PYTHON CODE TEMPLATES

### Template 1: Trade Setup Generator

```python
def generate_nirmal_setup(ticker, current_price, support_level, portfolio=100000):
    entry_1 = current_price
    entry_2 = current_price * 0.92
    target_1 = current_price * 1.12
    target_2 = current_price * 1.25
    stop_loss = min(entry_2 * 0.92, support_level * 0.98)
    
    risk_per_share = entry_1 - stop_loss
    max_risk = portfolio * 0.02
    shares = int(max_risk / risk_per_share)
    
    max_position = portfolio * 0.05
    if shares * entry_1 > max_position:
        shares = int(max_position / entry_1)
    
    reward = (target_2 - entry_1) / entry_1 * 100
    risk = (entry_1 - stop_loss) / entry_1 * 100
    ratio = reward / risk
    
    return {
        'ticker': ticker,
        'entry_1': round(entry_1, 2),
        'entry_2': round(entry_2, 2),
        'target_1': round(target_1, 2),
        'target_2': round(target_2, 2),
        'stop_loss': round(stop_loss, 2),
        'shares': shares,
        'risk_reward_ratio': round(ratio, 2),
        'status': 'GOOD' if ratio >= 2 else 'POOR'
    }
```

### Template 2: Position Monitor

```python
def monitor_position(ticker, current_price, entry_1, entry_2, target_1, target_2, stop_loss):
    entry_avg = (entry_1 + entry_2) / 2
    
    if current_price <= stop_loss:
        return {
            'action': 'STOP_LOSS',
            'exit_price': stop_loss,
            'return_pct': ((stop_loss - entry_avg) / entry_avg) * 100,
            'reason': 'MANDATORY EXIT'
        }
    
    if current_price >= target_2:
        return {
            'action': 'EXIT_ALL',
            'exit_price': target_2,
            'return_pct': ((target_2 - entry_avg) / entry_avg) * 100,
            'reason': 'Target 2 hit'
        }
    
    if current_price >= target_1:
        return {
            'action': 'SELL_50%',
            'exit_price': target_1,
            'return_pct': ((target_1 - entry_avg) / entry_avg) * 100,
            'reason': 'Target 1 hit'
        }
    
    return {'action': 'HOLD', 'reason': 'Between entry and targets'}
```

def calculate_portfolio_risk(positions):

    largest = max(positions, key=lambda x: x['position_pct'])
    
    return {
        'total_positions': len(positions),

        'largest_position': largest['ticker'],
        'status': 'OK' if total_risk_pct < 0.10 else 'REDUCE SIZES'
